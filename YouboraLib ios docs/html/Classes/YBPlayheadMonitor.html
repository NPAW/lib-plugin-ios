Overview

This class periodically checks the player’s playhead in order to infer buffer and/or seek events.

The PlayheadMonitor is bounded to a YBPlayerAdapter and fires its buffer and seek start and end methods.

In order to use this feature, monitorPlayhead should be called from the Adapter’s overridden constructor.
